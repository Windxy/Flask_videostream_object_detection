# Flask_videostream_object_detection
Flask做实时视频流处理并进行coco目标检测，一个简单的案例

## Requirement

```bash
Flask==1.1.2
greenlet==0.4.15
numpy==1.23.1
onnxruntime==1.10.0
opencv_python==4.5.2.52
```

## run

step1. 
```bash
python app.py
```
step2.
然后，打开网页链接 http://127.0.0.1:9011/
